// Конспект по разным темам

Вызовите компилятор g++ и компоновщик, используя следующую командную строку:
g++ -о hello Hello.срp
Эта команда приказывает компилятору g++ создать выполнимый файл h e llo 
путем компиляции исходного файла C++ H e llo . срр.

Инициализация списком выглядит следующим образом: 
int largeNum = 5000000;
short anotherNum{ largeNum }; // Ошибка сужения!
int anotherNum{ largeNum }; // OK!
float someFloat{ largeNum }; // Ошибка! Возможно сужение
float someFloat{ 5000000 }; // OK! 5000000 помещается в float

Автоопределение типа переменной
auto coinFlippedHeads = true;

Константа 
Ключевое слово co n stex p r позволяет объявлять константы подобно функциям:
constexpr double GetPi() {return 3.1415926;}

Одно co n stex p r-выражение может использовать другое:
constexpr double TwicePiO {return 2 * GetPi();}

c o n s te x p r может выглядеть как функция, однако обеспечивает возможности 
оптимизации с точки зрения компилятора и приложения. До тех пор, пока компилятор в
состоянии вычислять константное выражение как конкретное значение, оно может
использоваться в инструкциях и выражениях везде, где ожидается константа
